WHY HTML IS SO IMPORTANT

Only with HTML (HyperText Markup Language), one cannot achieve everything what a webpage looks like. We need to integrate javascript for easier convenience, error handling. JQuery also helps in this context. css (cascading style sheet) for beautification. For basic designing, above 3 is enough and even we can present nicer UX , error validation and presentation. AJAX(Asynchronous JavaScript and XML) will be the next target and user interactive responses can be brought more which makes web page user friendly. Latest technologies like node js, angular js etc are more available in the market. On;ine tutorials are also available lot. By mastering the required, one can build nicer pages and of course with the base of HTML. May be we can conclude HTML as forefather for web page creation.
